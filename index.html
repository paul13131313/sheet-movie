GOOGLE SHEETS VIDEO OVERLAY

Google Sheetsのセルをピクセルに見立てて、動画をモザイク状にリアルタイム再生するChrome拡張機能。スプレッドシートが映像キャンバスに変わる。

Download ZIP

How It Works

01

拡張機能をインストール

ZIPをダウンロードして展開。Chromeの拡張機能管理画面で「パッケージ化されていない拡張機能を読み込む」からフォルダを選択。

02

スプレッドシートを開く

Google Sheetsで任意のスプレッドシートを開くと、右上にコントロールパネルが表示される。

03

動画を選択して再生

mp4ファイルを選択して ▶ を押すだけ。セル1つ1つがピクセルとなり、モザイク状の映像がスプレッドシート上で動き出す。

Features

🎬

セル単位モザイク

Sheetsのセルグリッドに完全同期したピクセル化描画

🔍

ズーム追従

50%〜200%のズームレベルに自動対応

🔊

音声再生

元動画の音声をそのまま再生。ミュート切替も可能

🖱️

操作を邪魔しない

ドロップダウンやメニューは映像の上に表示される

Install

Chrome Web Storeには未公開です。以下の手順で手動インストールしてください。

  1. ZIPをダウンロードして展開する
  2. Chromeで chrome://extensions を開く
  3. 右上の「デベロッパーモード」をONにする
  4. 「パッケージ化されていない拡張機能を読み込む」をクリック
  5. 展開したフォルダ（sheet-movie-main）を選択
  6. Google Sheetsを開くと自動で有効になる
